# Cybersecurity Audit for Family Service of the Piedmont

## Objective


The Cybersecurity Audit project aimed to evaluate and strengthen the security posture of a real organization through a structured assessment process. The primary focus was to analyze existing security controls, identify vulnerabilities, and assess risks in alignment with industry frameworks such as NIST CSF. This hands-on experience was designed to develop practical skills in risk analysis, security auditing, and delivering actionable recommendations to improve organizational defenses.

### Skills Learned


- Conducted comprehensive cybersecurity assessments of real organizations
- Applied risk analysis methodologies to identify and prioritize vulnerabilities
- Evaluated security controls using frameworks such as NIST Cybersecurity Framework (CSF)
- Developed the ability to translate technical findings into business-friendly recommendations
- Strengthened communication skills through stakeholder interviews and presentations
- Enhanced critical thinking in identifying security gaps and proposing mitigation strategies

### Tools Used


- NIST Cybersecurity Framework (CSF) for control evaluation
- Risk assessment and auditing methodologies
- Google Docs for documentation and report development
- Microsoft Excel for organizing assessment data
