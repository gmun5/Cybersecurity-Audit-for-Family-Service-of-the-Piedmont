The Cybersecurity Audit project aimed to evaluate and strengthen the security posture of a real organization through a structured assessment process. This hands-on experience was designed to develop practical skills in risk analysis, security auditing, and delivering actionable recommendations to improve organizational defenses.

- Strengthened communication skills through stakeholder interviews and presentations
- Enhanced critical thinking in identifying security gaps and proposing mitigation strategies
